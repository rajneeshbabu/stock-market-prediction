Project Overview

This project predicts stock closing prices using a Bidirectional LSTM neural network trained on historical price data enriched with 10 technical indicators.

Technical Indicators Used

✅ SMA 20 / SMA 50 (trend)
✅ EMA 12 / EMA 26 (trend)
✅ MACD + Signal (momentum)
✅ RSI 14 (overbought/oversold)
✅ Bollinger Band Width
✅ Daily Return & 20d Volatility
✅ 5-Day Price Momentum
✅ Volume Ratio (vs 20d avg)

Dataset

Price/Volume Data for All US Stocks & ETFs — Daily OHLCV data for all NYSE/NASDAQ/NYSE MKT stocks. Trained on AAPL 2015–2024. 80/10/10 train/val/test split.

This is an educational project. Stock price predictions are inherently uncertain. Do NOT use this for real investment decisions.
